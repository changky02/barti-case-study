Framework Overview - Embed Code

Use this code to embed the interactive 6-Stage PM Framework into your presentation or website.

Option 1: Simple iframe Embed (Recommended)

If you host the HTML file on a web server, use this iframe code:

<iframe
  src="framework-overview.html"
  width="100%"
  height="800"
  frameborder="0"
  style="border: none; border-radius: 8px;"
></iframe>
Note: Replace framework-overview.html with the actual URL where you host the file (e.g., https://yoursite.com/framework-overview.html)

Usage Instructions

For Gamma:

  1. Add an "Embed" or "HTML" card to your slide
  2. Paste the Inline HTML code (Option 2)
  3. Adjust height if needed (recommended: 400-500px)

For PowerPoint/Keynote:

  1. Save framework-overview.html to your web server
  2. Insert → Web Object/iframe
  3. Use the iframe embed code (Option 1) with full URL

For Google Slides:

  1. Take a screenshot of the framework
  2. Insert as image (Google Slides doesn't support interactive HTML)
  3. Or link to the hosted HTML page
